Wprowadzenie do styli - ćwiczenie 1

Kaskadowe Arkusze Stylów to język służący do opisu formy prezentacji stron WWW


Style można dołączać na trzy sposoby
  • jako osobny plik - tzw. zewnętrzny arkusz CSS
  • osadzając je między tagami <style> i </style> koniecznie umieszczonymi wewnątrz sekcji <head>
  • umieszczając je wewnątrz atrybutu znacznika - tzw style inline

Zewnętrzny arkusz tylów zawiera CSS w oddzielnym pliku z .cssrozszerzeniem. Jest to najczęstsza i najprzydatniejsza metoda wprowadzania CSS do dokumentu. Możesz połączyć pojedyńczy plik CSS z wieloma stronami internetowymi, stylizując je wszystkie tym samym arkuszem stylów CSS.
Odwołanie do zewnętrznego arkusza stylów CSS odbywa się za pomocą tagu <link> z atrybutem href
<link rel="stylesheet" href="nazwa akusza wraz ze scieżką dostępu">

Wewnętrzny arkusz stylów znajduje się w dokumencie HTML
Aby utworzyć wewnętrzny arkusz stylów, należy umieszcić CSS wewnątrz <style> elementu zawartego w HTML <head>

Style inline to deklaracje CSS, które wpływają na pojedyńczy element HTML, zawarty w atrybucie style.
<p style="color:red;">This is my first CSS example</p>
